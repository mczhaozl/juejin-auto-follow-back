# 更新日志

## [2.0.0] - 2026-02-10

### 🎁 新增功能：五倍返回

- **多账号支持**：支持主账号 + 4个小号，实现五倍返回效果
- **智能回关**：主账号获取粉丝列表，所有账号自动回关
- **灵活配置**：可以只配置部分小号，支持1-5个账号任意组合
- **详细日志**：每个账号的执行情况都会单独记录

### 🔧 技术改进

- 重构 `JuejinFollowBot` 类，支持账号名称标识
- 自动从 Cookie 中提取 UUID，无需手动配置
- 优化日志输出，清晰显示每个账号的执行状态
- 添加账号间延时，避免操作过于频繁

### 📝 文档更新

- 新增 `config/multi-account-setup.md` - 多账号配置详细指南
- 新增 `USAGE.md` - 五倍返回使用指南
- 新增 `test_multi_account.sh` - 本地测试脚本
- 更新 `README.md` - 添加五倍返回说明

### 🎯 使用方式

#### 单账号模式（原有功能）
只配置 `JUEJIN_COOKIES`，正常自动回关

#### 多账号模式（新功能）
配置多个 Cookie：
- `JUEJIN_COOKIES` - 主账号（必需）
- `JUEJIN_COOKIES_ACCOUNT2` - 小号1（可选）
- `JUEJIN_COOKIES_ACCOUNT3` - 小号2（可选）
- `JUEJIN_COOKIES_ACCOUNT4` - 小号3（可选）
- `JUEJIN_COOKIES_ACCOUNT5` - 小号4（可选）

### 📊 执行效果

```

📋 发现 3 位新粉丝待回关

[主账号] 成功回关 3 人
[小号1] 成功回关 3 人
[小号2] 成功回关 3 人
[小号3] 成功回关 3 人
[小号4] 成功回关 3 人

总计：15 次成功关注（5倍返回）
```

### ⚠️ 注意事项

1. Cookie 需要定期更新（约30天过期）
2. 建议不要设置过高的执行频率
3. 仅用于自动回关粉丝，遵守社区规范
4. 小号可以只配置部分，支持任意组合
