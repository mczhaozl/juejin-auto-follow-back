# 油猴脚本入门：下载安装与好用脚本推荐

> 一文搞懂 Tampermonkey（油猴）的下载、安装与使用，并推荐一批实用脚本，让你在浏览器里解锁去广告、网页增强、一键解析等能力。

## 一、背景 / 为什么用油猴

**油猴**（Tampermonkey /  Greasemonkey）是一种**用户脚本管理器**：安装后，你可以为指定网站注入自己或他人写的 JavaScript，在不改网站源码的前提下，实现去广告、改样式、抓数据、一键翻译等。

- **场景**：某网站广告太多、排版不顺眼、缺少「一键下载」；又不想等官方改，也不想每次都手动操作。
- **做法**：装一个脚本，刷新即生效，关掉脚本即恢复原样，可随时安装/卸载，灵活可控。

常见用途：去广告、网页样式美化、视频/音频解析下载、购物比价、自动签到、翻译与阅读辅助等。

---

## 二、下载与安装

油猴本身是一个**浏览器扩展**，需先安装扩展，再在扩展里「添加脚本」或从脚本站安装具体脚本。

### 1. 安装 Tampermonkey 扩展

| 浏览器     | 安装方式 |
|------------|----------|
| **Chrome** | [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) 搜索 Tampermonkey，点击「添加至 Chrome」 |
| **Edge**   | [Microsoft Edge 加载项](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) 搜索 Tampermonkey，获取扩展 |
| **Firefox**| [Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/) 安装 Tampermonkey |

安装后，浏览器工具栏会出现 Tampermonkey 图标；首次使用可能需刷新页面或重启浏览器。

### 2. 安装具体脚本的两种方式

- **从脚本站安装（推荐）**  
  打开 [Greasy Fork](https://greasyfork.org/zh-CN)、[OpenUserJS](https://openuserjs.org/) 等，搜索脚本名或关键词，进入脚本页后点击「安装此脚本」，Tampermonkey 会弹出编辑页，确认后保存即可。
- **手动新建**  
  点击 Tampermonkey 图标 →「管理面板」→「+」创建新脚本，把脚本代码粘贴进去，保存。脚本头部的 `@match` / `@include` 决定在哪些网址生效。

安装完成后，访问脚本所匹配的网站时，脚本会自动运行（可在管理面板中启用/禁用）。

## 三、好用的油猴脚本推荐

以下按**类别**列举一些常用、口碑较好的脚本（名称与链接以脚本站当前为准，安装前请看清适用站点与更新日期）。

### 1. 去广告与页面精简

- **uBlock 类**：若你已用 uBlock 等扩展，可不再重复；若只想轻量去广告，可搜「AD 屏蔽」「广告过滤」类脚本，针对具体站点去广告条、弹窗。
- **知乎/简书等去广告**：在 Greasy Fork 搜索「知乎 去广告」「简书 净化」，可去掉首页推广、侧栏广告等，让页面更干净。

### 2. 网页增强与阅读体验

- **网页限制解除**：如「解除 B 站区域限制」「解除复制限制」等，可搜「解除限制」「right click」等关键词，按需安装（注意合规与版权）。
- **目录/大纲**：为长文页面生成目录，方便跳转，可搜「目录」「TOC」。
- **阅读模式**：将正文提取为简洁排版，可搜「阅读模式」「Reader」。

### 3. 视频 / 音频与下载

- **视频解析 / 下载**：如 B 站、YouTube 等站的解析与下载脚本，在 Greasy Fork 搜索「B站 下载」「YouTube 下载」等，安装后页面上会出现下载按钮或解析链接（注意遵守平台规则与版权）。
- **音频直链**：部分站点会提供「获取音频直链」类脚本，便于在播放器或下载工具中使用。

### 4. 购物与比价

- **历史价格**：在电商商品页显示价格走势、历史最低价（脚本站搜「价格」「比价」）。
- **促销提醒**：到价提醒、券提醒等，可按平台名 + 「优惠」搜索。

### 5. 效率与自动化

- **自动签到**：部分社区、论坛有一键签到脚本，可搜站点名 + 「签到」。
- **网盘直链 / 加速**：针对某网盘的直链解析、下载加速等（需注意平台规则）。

使用建议：**一次不要装太多**，先装 1～2 个刚需的，确认稳定再逐步添加；优先选**安装量大、近期有更新、评论区反馈好**的脚本。

---

## 四、使用注意点

- **安全**：脚本拥有页面内 JS 权限，可能读取表单、Cookie 等，只从可信站点（如 Greasy Fork、OpenUserJS）安装，并尽量选维护活跃的脚本。
- **兼容**：部分脚本针对旧版页面编写，网站改版后可能失效，可关注脚本更新或评论区。
- **合规与版权**：下载、解析类脚本请仅用于个人学习与合理使用，勿用于传播盗版或违反平台条款。

---

## 总结

- **油猴** = 浏览器扩展 + 用户脚本：先装 Tampermonkey，再装具体脚本；脚本按 `@match` 在指定网站自动运行。
- **下载**：Chrome / Edge / Firefox 从各自扩展商店安装 Tampermonkey；脚本从 Greasy Fork 等站「安装此脚本」即可。
- **好用脚本**：按需选择去广告、阅读增强、视频/音频解析、比价、签到等类别，优先选安装量大、近期更新的脚本，并注意安全与合规。

如果你有特别想优化的网站或场景，可以在脚本站用站点名 + 功能关键词搜索，一般都能找到现成方案；找不到时再考虑自己写一段简单脚本，用 Tampermonkey 管理即可。
