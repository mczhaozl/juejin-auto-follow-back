# 2026 年 2-3 月免费 AI 账号与 Token 额度汇总：开发者薅羊毛指南

> 汇总 2026 年 2-3 月主流 AI 免费 API 额度与试用 credits，海外+国内，附薅羊毛顺序。以官网为准。

---

## 一、为什么要汇总：免费额度能省多少

各家为了拉新和生态，会提供**注册即得的免费 credits** 或**长期免费 tier**。把多个平台的额度加起来，不花钱也能拿到**上百美元等价**的调用量，足够做小项目、跑 demo、学习 API。本文按「海外」和「国内」分块整理，并标出**是否需要绑卡、是否限时**，方便你按自己的需求选。

---

## 二、海外：OpenAI、Claude、Gemini、Groq、Mistral、DeepSeek、xAI

以下为 2026 年初常见信息，**具体以各官网最新说明为准**。

### 2.1 OpenAI

- **新用户试用**：新账号通常有 **$5 免费额度**，无需绑卡，一般 **3 个月内有效**；此前更慷慨的长期免费试用已在 2025 年中结束。
- **创业/研究者**：Startup 计划可申请 **最高约 $100,000** API 额度；研究者计划（Model Index for Researchers）符合条件的项目可申请 **约 $1,000** 额度。
- **参考**：[OpenAI 定价](https://openai.com/api/pricing/)、[研究者计划](https://platform.openai.com/docs/model-index-for-researchers)。

### 2.2 Anthropic（Claude）

- **新用户**：注册并完成手机验证后，一般有 **约 $5 的免费 API credits**，可用于调用 Claude 模型。
- 具体额度与有效期以 [Anthropic 官网](https://www.anthropic.com/) 或 Console 说明为准。

### 2.3 Google Gemini

- **免费层**：**Gemini** 提供**按速率限制的免费访问**，无需绑卡即可使用；适合开发与原型。
- **长上下文**：Google AI Studio 中 **Gemini 2.5 Pro** 等支持约 **100 万 token 上下文**，免费层也可体验。
- 参考：[Google AI Studio](https://aistudio.google.com/)。

### 2.4 Groq

- **免费 API**：无需信用卡即可使用，免费层可调用 **Llama 3.3 70B、Llama 4 Scout、Qwen3** 等；约 **30–60 次/分钟**，70B 模型约 **1,000 次/天**。
- **特点**：LPU 推理快，适合对延迟敏感的原型。
- 参考：[Groq Cloud](https://console.groq.com/)。

### 2.5 Mistral AI

- **免费 API**：**Mistral Large、Small、Codestral** 等可免费调用；约 **2 次/分钟** 速率限制，**每月约 10 亿 token** 额度，无需绑卡。
- 参考：[Mistral 官网](https://mistral.ai/)。

### 2.6 DeepSeek

- **新用户**：**约 500 万 token 免费**，无需绑卡；用完后按量计费，定价较低（如 V3.2 约 $0.27/M input）。
- 参考：[DeepSeek 平台](https://platform.deepseek.com/)。

### 2.7 xAI（Grok）

- **新用户**：截至 2026 年初，注册可获得 **约 $25 免费 credits**；另有 **数据共享计划** 可再获约 **$150/月** 额度（具体以官网为准）。
- **特点**：Grok 与 X 整合、长上下文（如 200 万 token），适合需要实时/社交数据的场景。
- 参考：[xAI 开发者](https://x.ai/)。

### 2.8 其它（Together AI 等）

- **Together AI**：无长期免费 tier，需至少 **$5 起充** 才能使用 Llama 4、DeepSeek R1 等；适合需要动态速率、稳定生产的场景。

## 三、国内：通义、豆包、智谱、文心

以下为 2026 年初常见信息，**以各平台最新活动为准**。

### 3.1 阿里云通义千问（百炼）

- **新用户**：可在百炼平台领取 **最高约 7,000 万 token** 免费额度；部分模型每个可领 **约 100 万 token**。
- 参考：[阿里云百炼](https://www.aliyun.com/product/bailian)、[百炼免费额度说明](https://developer.aliyun.com/article/1690975)。

### 3.2 字节豆包

- **永久免费额度**：各模型通常有 **约 50 万 token 永久免费**（不过期）；多模型合计可达 **约 300 万 token**；曾有过额外赠送数亿 token 的活动，当前以官网为准。
- 参考：[豆包开放平台](https://www.volcengine.com/product/doubao)。

### 3.3 智谱（GLM / 智谱清言）

- **免费层**：注册即可用，**每日约 50 次** 交互额度（具体以产品页为准）。
- **教育优惠**：高校/科研团队可申请 **约 1 亿 token/年** 免费额度。
- 参考：[智谱开放平台](https://open.bigmodel.cn/)、[智谱定价](https://www.zhipucn.com/price.html)。

### 3.4 百度文心

- **免费开放**：**ERNIE Speed、ERNIE Lite** 等模型已**免费开放** API 调用，无需单独领额度即可使用。
- 参考：[百度智能云千帆](https://cloud.baidu.com/product/wenxinworkshop)。

## 四、薅羊毛姿势：怎么领、怎么用

### 4.1 注册顺序建议

- **先领「无需绑卡」的**：Gemini、Groq、Mistral、DeepSeek、国内通义/豆包/文心等，零成本先跑通。
- **再领「送 credits」的**：OpenAI $5、Claude 约 $5、xAI 约 $25 等，按需绑卡或仅用赠送额度。
- **有项目再申「大额」**：OpenAI Startup、研究者计划、智谱教育优惠等，需要材料或审核，适合有明确用途时再申。

### 4.2 使用注意

- **有效期**：OpenAI 等试用 credits 常有** 3 个月**等限制，过期不用会作废。
- **速率与配额**：免费层多为**限 QPS/日请求/月 token**，做原型够用，上生产要看好限流与计费。
- **合规**：仅限个人学习、开发测试或平台允许的用途，遵守各平台服务条款与合规要求。
