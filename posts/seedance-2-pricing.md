# 1 秒 1 元：字节 Seedance 2.0 模型 API 定价公布

> 火山引擎公布 Seedance 2.0 视频生成 API 按量计费：两档单价、折合约 1 元/秒，开发者可按 token 估算成本并接入方舟平台。

## 一、变动主题与一句话概括

**字节跳动**通过**火山引擎方舟大模型服务平台**正式公布 **Seedance 2.0** 视频生成模型的 **API 定价**。按**百万 tokens** 计费，分「含视频输入」与「纯文本/图像生成」两档；按公开换算，生成约 15 秒视频单条成本约 **15 元**，被媒体概括为「**1 秒 1 元**」，成为国内 AI 视频领域首个清晰可落地的商业化计费标尺之一。

## 三、定价标准（对开发者的影响）

### 计费方式

Seedance 2.0 API 采用**按量计费**，以 **tokens 消耗** 为计费单位，与分辨率、输入类型相关：

| 计费类型         | 价格（元/百万 tokens） | 典型场景           |
|------------------|------------------------|--------------------|
| 含视频输入（V2V）| 28                     | 视频编辑、续写等   |
| 不含视频输入（T2V/图生视频等） | 46             | 纯文本/图像生成   |

即：**视频编辑类请求更便宜**（约 28 元/百万 tokens），**纯生成更贵**（46 元/百万 tokens）。

### 「1 秒 1 元」怎么来的

媒体报道中的换算大致为：生成约 **15 秒** 视频，约消耗 **30.888 万 tokens**（以纯生成档 46 元/百万 tokens 计），单条成本约 **15 元**，折合 **约 1 元/秒**。因此「1 秒 1 元」是**按 15 秒视频、纯生成档换算后的通俗说法**，实际费用仍以 **tokens × 单价** 为准，不同时长、分辨率、输入类型会不同。

### 开发接入要点

- **计费维度**：按 **tokens** 计费，不是按「条」或「秒」；调用前可根据文档估算单次请求的 token 消耗。
- **成本控制**：需要控制成本时，可优先考虑**视频编辑（V2V）** 等含视频输入的调用，单价更低。
- **接入方式**：通过**火山引擎方舟**创建模型单元、获取 API Key，按 [Seedance 2.0 API 参考](https://www.volcengine.com/docs/82379/1393047) 与 [SDK 示例](https://www.volcengine.com/docs/82379/1366799) 集成；具体开放区域与限额以控制台与文档为准。

## 四、Seedance 2.0 简要说明

Seedance 2.0 是字节 **Seed 团队**推出的多模态 **AI 视频生成模型**，支持：

- **多模态输入**：文本、图像、视频、音频均可作为参考或驱动；
- **能力**：文生视频、图生视频、视频编辑/续写、角色一致性、多镜头叙事等；
- **输出**：支持 1080p 及以上分辨率（如 2K 等以文档为准）。

适合需要「导演级」可控性与一致性的视频生成、短剧、营销素材等场景。
