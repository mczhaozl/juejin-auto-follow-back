# 掘金文章选题与已写记录（Outline）

按**大主题**分文件夹（英文名），每个主题下 `index.md` 记录「已写过的选题」，便于：

- 写**续集**或同主题深化时不重复
- 规划**不同主题**的下一篇
- 快速查阅某领域已覆盖内容

## 目录约定

- 一级目录：**大主题**，英文文件夹名。
- 示例：`Css`、`javascript`、`reactApi`、`Vue`、`webpack`、`Node`、`Go`、`PostgreSQL`、`Android`、`workplace-story`（职场小故事：介绍第三方工具或 API 踩坑）等；可按需新增。
- 每个主题下：`index.md`，用列表记录已写选题（标题或关键词 + 可选 databae 路径/日期）。

## index.md 格式示例

```markdown
# 主题名（如 CSS）已写选题

- flex（Flex 布局 / 弹性盒）
- 滚动驱动动画 scroll-timeline
```

新增文章后，在对应主题的 `index.md` 中追加一条记录。
